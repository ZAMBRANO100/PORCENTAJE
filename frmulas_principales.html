FÓRMULAS PRINCIPALES

Calcular el porcentaje de una cantidad

  • Para hallar el P% de un número 𝑁, se multiplica el decimal del porcentaje por la cantidad.

  • Fórmula: Porcentaje=(𝑃/100)×𝑁
    Ejemplo: Hallar el 15% de 80.
    (15/100)×80=0.15×80=12

Hallar un número a partir de un porcentaje

  • Si sabemos que un porcentaje 𝑃 equivale a una cantidad 𝐶, podemos encontrar el total 𝑇 con una regla de tres o una fórmula.
    Fórmula:
    𝑇=(𝐶/𝑃)×100

Ejemplo: Si el 20% de un número es 50, ¿cuál es el número?
𝑇=(50/20)×100=2.5×100=250

Calcular qué porcentaje representa una cantidad de otra

  • Para saber qué porcentaje representa una cantidad 𝐶 de un total 𝑇, se divide la parte entre el total y se multiplica por 100.
    Fórmula:
    Porcentaje=(𝐶𝑇)×100

Ejemplo: ¿Qué porcentaje de 200 representa 40?
(40/200)×100=0.2×100=20%
